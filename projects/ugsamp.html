SAMP Server

  • Tech Stack: Pawn, MySQL, C++
  • Github URL: Project Link

This project was created by me to allow players to play the popular multiplayer platform, SAMP in the UnionGamers community. This Script, was written in Pawn, using C++ and C and some libraries from the community.

This server, was developed in a modular programming style, and include a lot of features and models to make it a great server. The script allow users to register, login, manage their characters, and play in a roleplay environment.
